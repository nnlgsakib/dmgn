# Plan 03-01: Core Network Package + Config Extensions

<objective>
Create the `pkg/network/` package that wraps go-libp2p into a clean Host abstraction with DHT and mDNS discovery, connection management, and peer listing. Extend config with networking fields. This is the foundational networking layer that CLI commands will consume.
</objective>

<must_haves>
- libp2p host created with HKDF-derived ed25519 peer identity
- DHT discovery with custom DMGN protocol prefix
- mDNS local discovery with configurable service tag
- Connection manager with configurable watermarks
- TCP and QUIC transports enabled
- NAT traversal via NATPortMap + autorelay
- Config struct extended with networking fields
- Unit tests for host lifecycle and peer identity derivation
</must_haves>

<task id="2">
<title>Create Network Host Package</title>

<read_first>
- pkg/identity/identity.go (DeriveKey method)
- internal/config/config.go (Config struct with new fields)
</read_first>

<action>
Create `pkg/network/host.go` with:

1. **`HostConfig` struct** — configuration for creating a host:
```go
type HostConfig struct {
    ListenAddrs    []string
    BootstrapPeers []string
    MDNSService    string
    MaxPeersLow    int
    MaxPeersHigh   int
    PrivateKey     crypto.PrivKey // libp2p crypto key
}
```

2. **`Host` struct** — wraps libp2p host with discovery:
```go
type Host struct {
    host       host.Host
    dht        *dht.IpfsDHT
    mdnsService mdns.Service
    ctx        context.Context
    cancel     context.CancelFunc
}
```

3. **`DeriveLibp2pKey(identity *identity.Identity) (crypto.PrivKey, error)`** — derives libp2p ed25519 key from DMGN identity using HKDF:
   - Call `identity.DeriveKey("libp2p-host", 32)` to get 32-byte seed
   - Use `ed25519.NewKeyFromSeed(seed)` from stdlib crypto
   - Marshal the 64-byte ed25519 private key and convert via `crypto.UnmarshalEd25519PrivateKey(privKeyBytes)`
   - Return the libp2p private key

4. **`NewHost(cfg HostConfig) (*Host, error)`** — creates libp2p host with:
   - `libp2p.Identity(cfg.PrivateKey)`
   - `libp2p.ListenAddrStrings(cfg.ListenAddrs...)`
   - `libp2p.ConnectionManager(connmgr.NewConnManager(cfg.MaxPeersLow, cfg.MaxPeersHigh, connmgr.WithGracePeriod(time.Minute)))`
   - `libp2p.NATPortMap()`
   - `libp2p.EnableRelay()`
   - `libp2p.DefaultTransports` (TCP + QUIC)
   - `libp2p.DefaultSecurity` (TLS + Noise)
   - `libp2p.DefaultMuxers` (yamux + mplex)

5. **`(h *Host) Start() error`** — starts DHT bootstrap and mDNS discovery
6. **`(h *Host) Stop() error`** — graceful shutdown: close mDNS, close DHT, close host
7. **`(h *Host) ID() peer.ID`** — returns the host's peer ID
8. **`(h *Host) Addrs() []multiaddr.Multiaddr`** — returns listen addresses
</action>

<acceptance_criteria>
- `pkg/network/host.go` exists and contains `type Host struct`
- `pkg/network/host.go` contains `func DeriveLibp2pKey`
- `pkg/network/host.go` contains `func NewHost`
- `pkg/network/host.go` contains `func (h *Host) Start() error`
- `pkg/network/host.go` contains `func (h *Host) Stop() error`
- `pkg/network/host.go` contains `DeriveKey("libp2p-host", 32)`
- `go build ./pkg/network/...` exits 0
</acceptance_criteria>
</task>

<task id="3">
<title>Create Discovery Manager</title>

<read_first>
- pkg/network/host.go
</read_first>

<action>
Create `pkg/network/discovery.go` with:

1. **DHT setup function** `setupDHT(ctx context.Context, h host.Host, bootstrapPeers []string) (*dht.IpfsDHT, error)`:
   - Parse bootstrap peers from multiaddr strings to `peer.AddrInfo`
   - Create DHT with `dht.New(ctx, h, dht.Mode(dht.ModeAutoServer), dht.ProtocolPrefix("/dmgn/kad/1.0.0"))`
   - If bootstrap peers provided, connect to each in a goroutine
   - Call `dht.Bootstrap(ctx)` to start DHT refresh
   - Return the DHT instance

2. **mDNS notifee** `type discoveryNotifee struct` implementing `mdns.Notifee`:
   - `HandlePeerFound(pi peer.AddrInfo)` — connects to discovered peer via host

3. **mDNS setup function** `setupMDNS(h host.Host, serviceTag string) (mdns.Service, error)`:
   - Create notifee
   - Create mDNS service with `mdns.NewMdnsService(h, serviceTag, notifee)`
   - Return service

Wire both into `Host.Start()` method.
</action>

<acceptance_criteria>
- `pkg/network/discovery.go` exists and contains `func setupDHT`
- `pkg/network/discovery.go` contains `func setupMDNS`
- `pkg/network/discovery.go` contains `type discoveryNotifee struct`
