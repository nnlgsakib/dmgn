   - m.Timestamp → pb.Timestamp
   - string(m.Type) → pb.Type
   - m.Embedding → pb.Embedding ([]float32 maps directly)

   - m.MerkleProof → pb.MerkleProof

   `func MemoryFromProto(pb *dmgnpb.Memory) *Memory` — converts proto Memory to Go Memory:
   - Reverse of above
   - pb.Type → Type(pb.Type)

   Add import: `dmgnpb "github.com/nnlgsakib/dmgn/proto/dmgnpb"`

2. In `pkg/memory/memory_test.go`, add roundtrip test `TestMemoryProtoRoundtrip`:
   - Create a Memory with all fields populated (including Embedding, Metadata, Links)
   - Convert to proto: `pb := mem.ToProto()`
   - Marshal to bytes: `data, _ := proto.Marshal(pb)`
   - Unmarshal from bytes: `pb2 := &dmgnpb.Memory{}; proto.Unmarshal(data, pb2)`
   - Convert back: `mem2 := MemoryFromProto(pb2)`
   - Assert all fields match original
   - Add import: `"google.golang.org/protobuf/proto"`

3. In `pkg/network/protocols.go`, add proto frame utilities alongside existing JSON ones:

   `func writeProtoFrame(w io.Writer, msg proto.Message, data []byte) error`:
   - `msgBytes, err := proto.Marshal(msg)`
   - Write 4-byte big-endian length of msgBytes
   - Write msgBytes
   - Write data if present
   - Same pattern as existing writeFrame, but proto.Marshal instead of json.Marshal

   `func readProtoFrame(r io.Reader, msg proto.Message, dataLen int) ([]byte, error)`:
   - Read 4-byte big-endian length
   - Read that many bytes into msgBytes
   - `proto.Unmarshal(msgBytes, msg)`
   - Read dataLen bytes if > 0
   - Return data bytes

   `func readProtoHeaderOnly(r io.Reader, msg proto.Message) error`:
   - Calls readProtoFrame with dataLen=0
   - Returns error only

   Add imports: `"google.golang.org/protobuf/proto"` and `dmgnpb "github.com/nnlgsakib/dmgn/proto/dmgnpb"`

4. Do NOT remove existing JSON writeFrame/readFrame — they are still used by handlers until Plan 02 migrates them.
