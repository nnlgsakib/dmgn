| proto DataLen field → io.ReadFull | Peer-controlled length used for memory allocation |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-1-04 | D (DoS) | RegisterStoreHandler | mitigate | Reuse existing maxShardLen (10MB) check on `req.DataLen` before allocating `make([]byte, req.DataLen)`. Proto int32 max is 2GB — must bound. |
| T-1-05 | T (Tampering) | RegisterStoreHandler | mitigate | Reuse existing SHA256 checksum validation after reading shard data — same as JSON path. |
| T-1-06 | S (Spoofing) | SendShard/FetchShard | accept | Peer identity verified by libp2p transport layer (TLS 1.3). Protocol layer does not need additional auth. |
</threat_model>
