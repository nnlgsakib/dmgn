<task id="06-01-03" title="Implement MCP tool handlers">
<read_first>
- pkg/mcp/server.go (just created)
- pkg/query/engine.go (QueryEngine.SearchLocal, BuildRequest)
- pkg/memory/memory.go (Memory struct, Create, Graph.TraverseFrom)
- pkg/storage/storage.go (Store.SaveMemory, GetRecentMemories, GetMemory, DeleteMemory)
- pkg/vectorindex/index.go (VectorIndex.Add, Remove)
- internal/crypto/crypto.go (Engine.Encrypt, Decrypt)
</read_first>
<action>
Create `pkg/mcp/tools.go` with all 7 tool handler implementations:

1. **handleAddMemory** — Create PlaintextMemory, encrypt, save to store, index embedding in vector index. Return `{id, timestamp, type}`.

2. **handleQueryMemory** — Use `QueryEngine.BuildRequest()` + `SearchLocal()`. If `include_content` is false, return snippets. If true, decrypt and return full content. Return `{results: [{memory_id, score, type, timestamp, content_or_snippet}], count}`.

3. **handleGetContext** — Get recent N memories via `store.GetRecentMemories(limit)`, decrypt each, format as structured context: `{memories: [{id, content, type, timestamp, metadata}], count, context_window_hint: "Recent memories from DMGN"}`.

4. **handleLinkMemories** — Use `store.AddEdge(from, to, weight, edgeType)`. Return `{from, to, edge_type, created: true}`.

5. **handleGetGraph** — Use `memory.Graph` + `TraverseFrom(startID, maxDepth)`. Return `{root_id, nodes: [{id, type, timestamp, link_count}], edges: [{from, to, type, weight}], depth}`.

6. **handleDeleteMemory** — Use `store.DeleteMemory(id)`, remove from vector index. Return `{id, deleted: true}`.

7. **handleGetStatus** — Collect stats from store, vector index, config. Return `{node_id, version, memory_count, vector_index_size, storage_path, config_summary}`.

Each handler follows the pattern:
```go
func (s *MCPServer) handleAddMemory(ctx context.Context, req *mcp.CallToolRequest, input AddMemoryInput) (*mcp.CallToolResult, AddMemoryOutput, error)
```
</action>
<acceptance_criteria>
- `pkg/mcp/tools.go` exists with 7 handler functions
- Each handler returns proper output struct
- `go build ./pkg/mcp/...` exits 0
</acceptance_criteria>
</task>
