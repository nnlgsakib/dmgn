### Task 2: Vector Index Package

**File:** `pkg/vectorindex/index.go`

Create a `VectorIndex` wrapping coder/hnsw with encrypted persistence:

```go
package vectorindex

import (
    "github.com/coder/hnsw"
)

// VectorIndex wraps an HNSW graph with encrypted persistence.
type VectorIndex struct {
    graph     *hnsw.Graph[string]  // key = memory_id
    dimension int                   // auto-detected from first vector, or from config
    encryptFn func([]byte) ([]byte, error)
    decryptFn func([]byte) ([]byte, error)
    indexPath string                // path to encrypted index file
    dirty     bool                  // true if index modified since last save
}

// NewVectorIndex creates a new empty vector index.
// encryptFn/decryptFn are for persisting the index encrypted.
// indexPath is the file path for encrypted index storage.
func NewVectorIndex(indexPath string, encryptFn, decryptFn func([]byte) ([]byte, error)) *VectorIndex

// Load loads an encrypted index from disk. Returns empty index if file doesn't exist.
func (vi *VectorIndex) Load() error

// Save encrypts and persists the index to disk.
func (vi *VectorIndex) Save() error

// Add inserts a vector for a memory ID. If dimension is 0, auto-detects from first vector.
func (vi *VectorIndex) Add(memoryID string, embedding []float32) error

// Search returns the top-k most similar memory IDs with scores.
func (vi *VectorIndex) Search(query []float32, k int) []SearchResult

// Remove removes a memory from the index.
func (vi *VectorIndex) Remove(memoryID string)

// Count returns the number of indexed vectors.
func (vi *VectorIndex) Count() int

// Dimension returns the detected embedding dimension (0 if empty).
func (vi *VectorIndex) Dimension() int
```

```go
// SearchResult holds a search result with memory ID and similarity score.
type SearchResult struct {
    MemoryID string
    Score    float64  // cosine similarity, 0.0 to 1.0
}
```

Implementation details:
- Use `hnsw.NewGraph[string]()` with `string` key = memory_id
- `Add()`: call `graph.Add(hnsw.MakeNode(memoryID, embedding))`
- `Search()`: call `graph.Search(query, k)`, convert to SearchResult
- `Save()`: `graph.Export(buf)` → encrypt buf → write file
- `Load()`: read file → decrypt → `graph.Import(reader)`
- `dimension` auto-detected from first `Add()` call. Subsequent `Add()` calls validate dimension matches.
- Thread safety: `sync.RWMutex` — read lock for Search, write lock for Add/Remove/Save/Load

### Task 3: Version Vector

**File:** `pkg/sync/vclock.go`

```go
package sync

// VersionVector tracks the latest sequence number from each known peer.
type VersionVector struct {
    entries map[string]uint64  // peer_id -> latest sequence number
    mu      sync.RWMutex
}

// NewVersionVector creates an empty version vector.
func NewVersionVector() *VersionVector

// Increment increments the local peer's sequence number and returns the new value.
func (vv *VersionVector) Increment(peerID string) uint64

// Get returns the sequence number for a peer (0 if unknown).
func (vv *VersionVector) Get(peerID string) uint64

// Merge updates the vector with another vector's entries (take max of each).
func (vv *VersionVector) Merge(other *VersionVector)

// MissingFrom returns peer entries where other has newer data than this vector.
// Used to determine what data to request from a peer.
func (vv *VersionVector) MissingFrom(other *VersionVector) map[string]uint64

// Marshal serializes the version vector to JSON bytes.
func (vv *VersionVector) Marshal() ([]byte, error)

// Unmarshal deserializes a version vector from JSON bytes.
func Unmarshal(data []byte) (*VersionVector, error)

// Clone returns a deep copy of the version vector.
func (vv *VersionVector) Clone() *VersionVector
```

**File:** `pkg/sync/vclock_store.go`

```go
// VClockStore persists version vectors in BadgerDB.
type VClockStore struct {
    db *badger.DB
}

// Save persists a version vector for the local peer.
func (s *VClockStore) Save(localPeerID string, vv *VersionVector) error

// Load retrieves the persisted version vector.
func (s *VClockStore) Load(localPeerID string) (*VersionVector, error)

// SaveSequence persists a memory's sequence tag.
func (s *VClockStore) SaveSequence(peerID string, seq uint64, memoryID string) error

// GetMemoriesAfter returns memory IDs with sequence > afterSeq for a given peer.
func (s *VClockStore) GetMemoriesAfter(peerID string, afterSeq uint64) ([]string, error)
```

Storage keys:
- `vv:{local_peer_id}` → JSON-encoded version vector
- `seq:{peer_id}:{zero-padded seq}` → memory_id (allows range scans)
