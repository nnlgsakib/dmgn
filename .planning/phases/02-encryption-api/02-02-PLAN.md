<objective>
Build a REST API server with JSON request/response bodies and HMAC-based authentication derived from the user's identity. Add a `dmgn serve` CLI command to start the API server.
</objective>

<must_haves>
- POST /memory endpoint accepts JSON body and stores encrypted memory
- GET /query?q=<text> endpoint searches memories and returns JSON results
- GET /status endpoint returns node status as JSON
- All endpoints require Authorization header with API key
- API key is deterministically derived from the user's ed25519 identity
- dmgn serve command starts the HTTP server
</must_haves>

</read_first>

<action>
Create `internal/api/auth.go`:

```go
package api

import (
    "crypto/hmac"
    "crypto/sha256"
    "encoding/hex"
    "net/http"
    "strings"
)

const (
    AuthHeader  = "Authorization"
    AuthScheme  = "Bearer"
    APIKeyPurpose = "api-key"
)

type AuthMiddleware struct {
    apiKeyHash []byte
}

func NewAuthMiddleware(identityDerivedKey []byte) *AuthMiddleware {
    h := sha256.Sum256(identityDerivedKey)
    return &AuthMiddleware{apiKeyHash: h[:]}
}

func DeriveAPIKey(identityDerivedKey []byte) string {
    return hex.EncodeToString(identityDerivedKey[:16])
}

func (a *AuthMiddleware) Authenticate(next http.Handler) http.Handler {
    return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
        authHeader := r.Header.Get(AuthHeader)
        if authHeader == "" {
            http.Error(w, `{"error":"missing authorization header"}`, http.StatusUnauthorized)
            return
        }

        parts := strings.SplitN(authHeader, " ", 2)
        if len(parts) != 2 || parts[0] != AuthScheme {
            http.Error(w, `{"error":"invalid authorization scheme, use Bearer"}`, http.StatusUnauthorized)
            return
        }

        providedKey, err := hex.DecodeString(parts[1])
        if err != nil {
            http.Error(w, `{"error":"invalid api key format"}`, http.StatusUnauthorized)
            return
        }

        providedHash := sha256.Sum256(providedKey)
        if !hmac.Equal(providedHash[:], a.apiKeyHash) {
            http.Error(w, `{"error":"invalid api key"}`, http.StatusUnauthorized)
            return
        }

        next.ServeHTTP(w, r)
    })
}
```

The API key is derived via `identity.DeriveKey("api-key", 0)` then truncated to 16 bytes and hex-encoded for the user to use as a Bearer token. The server stores only the SHA256 hash for comparison.
</action>

- Contains `func (a *AuthMiddleware) Authenticate`

```go
package api

type AddMemoryRequest struct {
    Content  string            `json:"content"`
    Type     string            `json:"type,omitempty"`
    Links    []string          `json:"links,omitempty"`
    Metadata map[string]string `json:"metadata,omitempty"`
}

type AddMemoryResponse struct {
    ID        string `json:"id"`
    Timestamp int64  `json:"timestamp"`
    Type      string `json:"type"`
}

type QueryRequest struct {
    Query string `json:"query,omitempty"`
    Limit int    `json:"limit,omitempty"`
}

type QueryResult struct {
    ID        string            `json:"id"`
    Content   string            `json:"content"`
    Type      string            `json:"type"`
    Timestamp int64             `json:"timestamp"`
    Links     []string          `json:"links"`
    Metadata  map[string]string `json:"metadata,omitempty"`
}

type QueryResponse struct {
    Results []QueryResult `json:"results"`
    Count   int           `json:"count"`
}

type StatusResponse struct {
    NodeID      string         `json:"node_id"`
    Version     string         `json:"version"`
    Storage     StorageStats   `json:"storage"`
    Network     NetworkStats   `json:"network"`
}

type StorageStats struct {
    MemoryCount int64  `json:"memory_count"`
    EdgeCount   int64  `json:"edge_count"`
    Path        string `json:"path"`
}

type NetworkStats struct {
    Status string `json:"status"`
    Peers  int    `json:"peers"`
}
```

Implement handlers:

1. `HandleAddMemory(w, r)` — POST /memory
   - Decode JSON body into AddMemoryRequest
   - Validate: content required, type defaults to "text"
   - Create PlaintextMemory, call memory.Create with encryptFn
   - Save via store.SaveMemory
   - Return 201 with AddMemoryResponse JSON

2. `HandleQuery(w, r)` — GET /query?q=<text>&limit=<n>
   - Extract query param `q` and optional `limit` (default 10)
   - If no query: return recent memories
   - Search via decryptFn like CLI query does
   - Return 200 with QueryResponse JSON

3. `HandleStatus(w, r)` — GET /status
   - Get store.GetStats()
   - Return 200 with StatusResponse JSON including node ID, version, storage stats
</action>
