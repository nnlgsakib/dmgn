### Task 2: Protocol Message Types

**File:** `pkg/network/protocols.go`

Define protocol IDs and message types:

```go
const (
    StoreProtocol = "/dmgn/memory/store/1.0.0"
    FetchProtocol = "/dmgn/memory/fetch/1.0.0"
)

type StoreRequest struct {
    MemoryID    string `json:"memory_id"`
    ShardIndex  int    `json:"shard_index"`
    TotalShards int    `json:"total_shards"`
    Threshold   int    `json:"threshold"`
    Checksum    string `json:"checksum"`
    DataLen     int    `json:"data_len"`
}

type StoreResponse struct {
    Status  string `json:"status"` // "ok" or "error"
    Message string `json:"message,omitempty"`
}

type FetchRequest struct {
    MemoryID   string `json:"memory_id"`
    ShardIndex int    `json:"shard_index"`
}

type FetchResponse struct {
    Status      string `json:"status"`
    MemoryID    string `json:"memory_id,omitempty"`
    ShardIndex  int    `json:"shard_index,omitempty"`
    TotalShards int    `json:"total_shards,omitempty"`
    Threshold   int    `json:"threshold,omitempty"`
    Checksum    string `json:"checksum,omitempty"`
    DataLen     int    `json:"data_len,omitempty"`
    Message     string `json:"message,omitempty"`
}
```

Message framing: 4-byte big-endian length prefix + JSON header + raw shard bytes (for store/fetch data).

Helper functions:
```go
func writeMessage(w io.Writer, header interface{}, data []byte) error
func readMessage(r io.Reader) (headerBytes []byte, data []byte, err error)
```

### Task 6: DHT Shard Provider Tracking

**File:** `pkg/network/shardrouter.go`

```go
type ShardRouter struct {
    dht  *dht.IpfsDHT
    host host.Host
}

// AnnounceShardProvider announces this peer as a provider for a shard.
func (r *ShardRouter) AnnounceShardProvider(ctx context.Context, memoryID string, shardIndex int) error

// FindShardProviders finds peers that hold a specific shard.
func (r *ShardRouter) FindShardProviders(ctx context.Context, memoryID string, shardIndex int, count int) ([]peer.AddrInfo, error)

// ShardCID generates a CID for a shard from memory ID and index.
func ShardCID(memoryID string, shardIndex int) cid.Cid
```

Uses `multihash.Sum()` with SHA2-256 on `memoryID:shardIndex` to create a CID, then `dht.Provide()` / `dht.FindProviders()`.
