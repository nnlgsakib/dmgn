## Tasks

### Task 1: Shard Distribution Manager

**File:** `pkg/sharding/distributor.go`

Orchestrates the full lifecycle: shard a memory → distribute shards → track providers.

```go
type Distributor struct {
    host        *network.Host
    store       *storage.Store
    router      *network.ShardRouter
    cfg         ShardConfig
}

func NewDistributor(h *network.Host, s *storage.Store, r *network.ShardRouter, cfg ShardConfig) *Distributor

// DistributeMemory shards a memory and sends shards to connected peers.
// Stores shards locally if insufficient peers. Returns distribution status.
func (d *Distributor) DistributeMemory(ctx context.Context, mem *memory.Memory) (*DistributionResult, error)

// ReconstructMemory fetches shards from peers and reconstructs the encrypted payload.
func (d *Distributor) ReconstructMemory(ctx context.Context, memoryID string) ([]byte, error)

type DistributionResult struct {
    MemoryID      string
    TotalShards   int
    Distributed   int   // shards sent to peers
    LocalOnly     int   // shards stored locally (pending)
    PeerMap       map[int]string // shard_index -> peer_id
}
```

### Task 3: Periodic Shard Audit

**File:** `pkg/network/rebalance.go`

```go
type ShardAuditor struct {
    distributor *sharding.Distributor
    router      *network.ShardRouter
    store       *storage.Store
    interval    time.Duration // default 5 minutes
    cancel      context.CancelFunc
}

func NewShardAuditor(d *sharding.Distributor, r *network.ShardRouter, s *storage.Store, interval time.Duration) *ShardAuditor

// Start begins periodic shard auditing in a goroutine.
func (a *ShardAuditor) Start(ctx context.Context)

// Stop cancels the periodic audit.
func (a *ShardAuditor) Stop()

// AuditOnce runs a single audit pass — checks all local memories for adequate shard distribution.
func (a *ShardAuditor) AuditOnce(ctx context.Context) (*AuditResult, error)

type AuditResult struct {
    MemoriesChecked   int
    ShardsRedistributed int
    Errors            []string
}
```

### Task 4: Wire into dmgn start

**File:** `internal/cli/start.go`

After creating the network host and before the signal wait:

1. Create `ShardRouter` from host DHT
2. Create `Distributor` with host, store, router, config
3. Register store/fetch protocol handlers on host
4. Create and start `RebalanceNotifiee` 
5. Create and start `ShardAuditor`
6. On shutdown: stop auditor, then host

### Task 6: Integration Tests

**File:** `tests/integration_test.go`

- `TestShardDistributeAndReconstruct` — Two nodes: Node A shards a memory, distributes to Node B, then reconstructs from fetched shards. Verify reconstructed payload matches original.
- `TestShardDistributionInsufficientPeers` — Single node: distribute shards, verify all stored locally with pending status.
- `TestProtocolStoreAndFetch` — Two nodes: store shard via protocol, fetch it back, verify data integrity.
- `TestShardRebalanceOnDisconnect` — Three nodes: distribute shards, disconnect one, verify rebalancing triggers re-replication to remaining peer.
