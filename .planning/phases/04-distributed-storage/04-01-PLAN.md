# Plan 04-01: Sharding Package + Shard Storage

**Phase:** 04-distributed-storage
**Wave:** 1 (parallel with 04-02)
**Depends on:** Phase 3 complete

## Objective

Create the sharding package (`pkg/sharding/`) implementing Shamir's Secret Sharing to split encrypted memory payloads into shards, and extend storage to persist shards in BadgerDB.

## Requirements Addressed

- **DIST-01:** Split memories into encrypted shards for distribution
- **DIST-04:** No single peer can reconstruct original data from shards alone

## Tasks

### Task 1: Shamir's Secret Sharing Implementation

**File:** `pkg/sharding/shamir.go`

Implement Shamir's Secret Sharing over GF(2^8):

```go
package sharding

// Split splits a secret byte slice into n shares with threshold k.
// Any k shares can reconstruct the secret; k-1 shares reveal nothing.
func Split(secret []byte, n, k int) ([][]byte, error)

// Combine reconstructs the original secret from k or more shares.
func Combine(shares [][]byte) ([]byte, error)
```

Implementation details:
- GF(2^8) arithmetic (add = XOR, multiply via log/exp tables)
- Random polynomial coefficients from `crypto/rand`
- Each share is `len(secret) + 1` bytes (1 byte for x-coordinate)
- Lagrange interpolation for reconstruction
- Input validation: `k >= 2`, `n >= k`, `n <= 255`, `len(secret) > 0`

### Task 2: Shard Manager

**File:** `pkg/sharding/sharding.go`

```go
type ShardConfig struct {
    Threshold  int  // k: minimum shares to reconstruct (default 3)
    TotalShards int // n: total shares to create (default 5)
}

type Shard struct {
    MemoryID   string `json:"memory_id"`
    ShardIndex int    `json:"shard_index"`
    TotalShards int   `json:"total_shards"`
    Threshold  int    `json:"threshold"`
    Data       []byte `json:"data"`
    Checksum   string `json:"checksum"` // SHA256 of Data
}

// ShardMemory splits a memory's encrypted payload into shards.
func ShardMemory(mem *memory.Memory, cfg ShardConfig) ([]Shard, error)

// ReconstructPayload reconstructs the encrypted payload from k+ shards.
func ReconstructPayload(shards []Shard) ([]byte, error)

// VerifyShard checks a shard's checksum integrity.
func VerifyShard(s Shard) bool
```

### Task 3: Shard Storage in BadgerDB

**File:** `pkg/storage/shards.go`

Extend storage.Store with shard operations:

```go
// SaveShard persists a shard to BadgerDB under key "shard:{memory_id}:{shard_index}"
func (s *Store) SaveShard(shard *sharding.Shard) error

// GetShard retrieves a specific shard by memory ID and index.
func (s *Store) GetShard(memoryID string, shardIndex int) (*sharding.Shard, error)

// GetShardsForMemory retrieves all locally stored shards for a memory.
func (s *Store) GetShardsForMemory(memoryID string) ([]*sharding.Shard, error)

// DeleteShard removes a shard from local storage.
func (s *Store) DeleteShard(memoryID string, shardIndex int) error

// GetShardStats returns shard storage statistics.
func (s *Store) GetShardStats() (map[string]int64, error)
```
